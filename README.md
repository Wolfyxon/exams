# Egzaminy (PL)
Moje repozytorium z rozwiązanymi egzaminami z programowania.

Tutaj znajdziesz zgrupowane rozwiązania egzaminów, ale mam też pojedyńcze projekty porozrzucane na innych repozytoriach (np. sama aplikacja konsolowa):
- https://github.com/Wolfyxon/android-for-school
- https://github.com/Wolfyxon/web-for-school
- https://github.com/Wolfyxon/cs-for-school
- https://github.com/Wolfyxon/python-for-school

**Ważne:** By uprościć przeglądanie, nie pakowałem projektów do zipa i nie układałem plików 1:1 tak jak być to powinno na egzaminie. W niektórych rozwiązaniach odpuściłem też se dokumentację środowiska i zrzuty ekranu.
